## Run Locally

Clone the project



Go to the project directory

```bash
  cd chitra_generator_backend/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon
```
Again go to the project directory

```bash
  cd chitra_generator_frontend/
```

Install dependencies

```bash
  npm install
```

Start the client

```bash
  npm start
```
